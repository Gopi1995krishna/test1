Migration tool
-------------------------

Migration is a command line tool to migrate data from Oracle Mobile Cloud Service (MCS) 18.4.3 to Oracle Mobile Hub (OMH) 19.1.3. 

The migration tool is provided as Python scripts with Dockerfile to build a Docker image with pre-configured environment.
The migration guide is available at https://docs.oracle.com/en/cloud/paas/mobile-cloud/mcsua/migrate-oracle-mobile-hub.html#GUID-35709E1F-CEBF-4FFC-BB07-562B1FDBB500.
